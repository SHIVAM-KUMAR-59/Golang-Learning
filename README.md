## 🔥 How to Run the Code

1. **Install [Go](https://go.dev/dl/) if you haven’t already.**
2. **Clone this repo:**
   ```bash
   git clone https://github.com/your-username/golang-learning.git
   cd golang-learning
   ```
3. **Run any Go file using:**
   ```bash
   go run filename.go
   ```
